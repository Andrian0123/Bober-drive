## 🚀 Быстрый старт (3 шага)

```
┌──────────────────────────────────────────────────────────────┐
│ ШАГ 1: Установите Ollama                                     │
└──────────────────────────────────────────────────────────────┘

Windows: https://ollama.ai/download/windows
Linux:   curl -fsSL https://ollama.ai/install.sh | sh
Mac:     brew install ollama


┌──────────────────────────────────────────────────────────────┐
│ ШАГ 2: Запустите Ollama сервер (отдельный терминал)         │
└──────────────────────────────────────────────────────────────┘

> ollama serve

[Оставьте это окно открытым]


┌──────────────────────────────────────────────────────────────┐
│ ШАГ 3: Запустите setup                                       │
└──────────────────────────────────────────────────────────────┘

Windows:    setup.bat
Linux/Mac:  ./setup.sh

[Выберите агента для установки]
```

## 💻 Команды управления

```
═══════════════════════════════════════════════════════════════
                    WINDOWS КОМАНДЫ
═══════════════════════════════════════════════════════════════

Простой запуск:
> launch-agent.bat hermes

Продвинутое управление (PowerShell):
> .\ollama-manager.ps1 list               # Список агентов
> .\ollama-manager.ps1 plugins            # Все плагины
> .\ollama-manager.ps1 plugins hermes     # Плагины агента
> .\ollama-manager.ps1 install hermes     # Установить
> .\ollama-manager.ps1 launch hermes      # Запустить
> .\ollama-manager.ps1 status             # Статус системы

Проверка установки:
> test-installation.bat


═══════════════════════════════════════════════════════════════
                   LINUX / MAC КОМАНДЫ
═══════════════════════════════════════════════════════════════

Управление:
$ ./ollama-manager.sh list                # Список агентов
$ ./ollama-manager.sh plugins             # Все плагины
$ ./ollama-manager.sh plugins hermes      # Плагины агента
$ ./ollama-manager.sh install hermes      # Установить
$ ./ollama-manager.sh launch hermes       # Запустить
$ ./ollama-manager.sh status              # Статус системы

Проверка установки:
$ ./test-installation.sh
```
