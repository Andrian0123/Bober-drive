## Быстрое использование

```python
from nexus_autonomous_daemon import create_autonomous_daemon

# Создать и запустить
daemon = create_autonomous_daemon(
    project_root="./my_project",
    vault_path="./my_project/.vault"
)
daemon.start()  # АВТОНОМНЫЙ РЕЖИМ

# Использовать
results = daemon.search("query")

daemon.stop()
```
